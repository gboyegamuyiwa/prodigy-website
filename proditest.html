What is ProdiTest App?

ProdiTest is a Computer Based Test (CBT) Web Application for Schools. It is a very simple CBT solutions developed in response to the yearnings of schools for a CBT Application that is affordable, functional, simple to use and easy to administered

Features of the App includes:

  • Multiple-Choice Questions and Answers
  • Mobile Responsiveness
  • Flexible Timer
  • Enable/Disable Registration
  • Results presented immediately
  • Download results into Microsoft Excel for printing
